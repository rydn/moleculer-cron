# 0.0.8 (2025-01-15)

## New
- Added explicit TypeScript test coverage for Moleculer 0.15 compatibility
- Added TypeScript support with `ts-jest` and type definitions validation
- Added Node.js 22+ requirement enforcement via `engines` field

## Fixes
- Verified compatibility with Moleculer 0.15 event-based handler pattern

## Breaking changes
- **Dropped support for Moleculer 0.14.x** - now requires Moleculer ^0.15.0
- **Minimum Node.js version increased to 22.x** (required by Moleculer 0.15)

## Changes
- Updated peer and dev dependencies to Moleculer ^0.15.0
- Enhanced Jest configuration to support TypeScript tests
- All existing cron job functionality remains compatible with 0.15
